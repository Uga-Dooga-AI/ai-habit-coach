# Feature Pack: Habit Creation

## Описание

Создание, редактирование и удаление привычек. Habit editor modal, тип привычки, частота, иконка/цвет.

## Файлы пака

- `happy-path.md`
- `edge-cases.md`
- `telemetry-notes.md`

## Adjacent areas

- Habit tracking (зависит от списка привычек)
- Streaks (привязаны к конкретным привычкам)
- AI coaching (персонализация по набору привычек)

# Feature Pack: Auth

## Описание

Firebase Auth: email/password, OAuth (если настроен), sign-out. Firebase UID используется для идентификации в Supabase.

## Файлы пака

- `happy-path.md`
- `edge-cases.md`
- `telemetry-notes.md`

## Adjacent areas

- Habit data (привязана к Firebase UID в Supabase)
- AI coaching (Supabase edge function авторизация через Firebase token)
- Analytics identity (Firebase UID → все провайдеры)

## Release-critical: Да
